# Forkline

Forkline is a lightweight Linux desktop client for Git repositories, including
private and self-hosted repositories. It uses the Git executable and credential
helpers already configured on the machine.

## Run

```bash
npm install
npm start
```

## Current features

- Open a local repository with a native folder picker.
- Inspect branches, remotes, commits, changed files and diffs.
- Stage and unstage individual files or all files.
- Commit, fetch, fast-forward pull and push.
- Switch branches and create branches.
- Refresh automatically when repository files change.

Forkline does not collect account data and does not impose repository hosting
or visibility restrictions. Authentication remains managed by Git, SSH and the
system credential helpers.
